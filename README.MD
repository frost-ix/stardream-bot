### ⚙️ 사용방법을 확인 해요

-   **'/사용방법'** 을 입력 시 봇의 사용방법을 안내 합니다.

### 💫 멤버 방송을 확인해요

-   3분 단위로 확인을 합니다.
-   **'/상태'** 를 입력하면 멤버 전원의 방송 온/오프라인을 확인합니다.
-   **'/상태 이름:[멤버 이름]'** 를 입력하면 입력 한 멤버'만' 방송 온/오프라인을 확인합니다.
